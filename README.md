# CS639 NLP Project

## contribution
I compiled the evaluation results from multiple defense runs into a consistent analysis format by extracting the key safety and utility metrics—BeaverTails, AdvBench, SST-2, and GSM8K—from heterogeneous JSON summaries, PDFs, and intermediate result files. I standardized these results into heatmap-ready and scatterplot-ready CSV files with shared fields for method, poison ratio, learning rate, epochs, metric, and score, which made the different defenses directly comparable across hyperparameter settings. I then used visualization analysis to move beyond raw tables: hyperparameter heatmaps showed how each defense behaved under increasingly extreme learning rates, epochs, and harmful-data ratios, while safety–utility scatter plots compared SST-2 performance against BeaverTails harmfulness to reveal the central tradeoff between preserving benign utility and resisting harmful fine-tuning. This data compilation and visualization work turned scattered experimental outputs into interpretable evidence for evaluating whether each defense remained robust under extreme fine-tuning conditions.
